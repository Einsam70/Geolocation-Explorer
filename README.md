# 1. Función

Permite obtener listados en formato CSV de rangos de IP de todos los paises con filtros por pais, región o ciudad. 

# 2. Como utilizar

Para que funcione correctame debe existir el archivo "**datos.csv**" dentro de la carpeta **csv** del proyecto. Este archivo lo puedes conseguir siguiendo las instrucciones del **archivo TXT** incluido en esa carpeta.

Una vez cumplido este requisito lanza index.htm (funciona localmente), selecciona el país, región o ciudad de la que deseas obtener el listado de IPs y por último pulsa en "Bajar CSV" con la información que precisas.

El archivo descargado contiene dos columnas separadas por coma (,) siendo el primer valor la IP inicial del rango y el segundo valor la IP final de ese rango.
